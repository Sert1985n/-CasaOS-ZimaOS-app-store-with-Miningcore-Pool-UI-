Папка для иконок приложений (App Store).
Положите сюда PNG/SVG с именами: btc.png, eth.png, doge.png, rvn.png, ltc.png, xmr.png, grs.png, dgb.png, bch.png, bc2.png и т.д.
В docker-compose можно указать icon как URL на raw GitHub этой папки, например:
https://github.com/USER/REPO/raw/main/Apps/img/btc.png

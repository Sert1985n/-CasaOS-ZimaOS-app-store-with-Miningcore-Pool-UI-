# Pool UI — public-pool-btc.ru

Все файлы лежат в папке **poolui**.

## Где что лежит

| Путь | Что это |
|------|--------|
| `C:\Users\WIN-10\poolui\` | **Корень проекта** — здесь лежат все файлы UI |
| `poolui\index.html` | Главная страница (шапка, меню, контейнер под контент) |
| `poolui\site.css` | Стили (темы, таблицы, карточки, кнопки) |
| `poolui\dashboard.css` | Стили графика (вертикальная полоса при наведении) |
| `poolui\app.js` | Логика SPA: роутер, API, графики, страницы |
| `poolui\assets\brand\` | Сюда при деплое копируют favicon.png и brand.png (логотип) |
| `poolui\assets\icons\` | Сюда при деплое копируют иконки монет (btc.png, bch.png и т.д.) |
| `poolui\deploy.sh` | Скрипт деплоя на Linux-сервер |

## Как открыть локально

1. Открыть папку в проводнике: **C:\Users\WIN-10\poolui**
2. Запустить любой простой HTTP-сервер из этой папки, например:
   - `npx serve -l 3000` (если установлен Node.js)
   - или открыть index.html через расширение «Live Server» в VS Code/Cursor
3. Открыть в браузере: http://localhost:3000 (или тот порт, что показал serve).

Данные подгружаются с `/api` — при локальном запуске нужен либо прокси на ваш бэкенд, либо тестовый API.

## Деплой на сервер

Скопируйте всю папку **poolui** на сервер и выполните там:

```bash
cd /путь/к/poolui
chmod +x deploy.sh
sudo ./deploy.sh
```

Картинки (favicon, логотип, иконки монет) берутся из папки, заданной в скрипте (по умолчанию `/media/ZimaOS-HD/pool-ui/img`).
